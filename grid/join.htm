Joining Braintec is easy. First you have to create an account and download the Humanupload Program. After you have installed this free software, your computer will automatically hook up into the BraintecGrid and begin working on a computing task. It's that simple!

 

System requirements
Check to ensure you have at least the minimum recommended system requirements:
- Windows XP, 2000, NT 4.0, Me, or 98
- Internet Explorer 5 of higher installed
- 64MB Memory (RAM) minimum
- Pentium 133 or above
- resolution 800 x 600 minimum to fit all the graphics on the screen

In Phase 2, we shall be able to support Mac OS X and Linux as well. These are not available yet, nor will they be around any time soon. But, stay tuned for news of new releases on these platforms.

Download and install
You can choose between an easy setup and a manual setup.

Easy setup:
Click the following link, then click the "open" button: SetupBraintecGrid.exe

Manual setup:
Right-click the following link and select "Save target as..." to save the screensaver in your windows folder: BraintecGrid.scr
Configure the screensaver in the Control Panel "Display" applet by selecting "BraintecGrid" as screensaver followed by apply.

 

Remove program
If you, for whatever reason, decide to stop joining the BraintecGrid, you can deactivate the "BraintecGrid" screensaver in the Control Panel "Display" applet. And you can completely remove the program by deleting the "BraintecGrid.scr" file in the Windows folder.
